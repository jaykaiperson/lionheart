# config/ — Централизованные настройки Lionheart

Всё, что ты хочешь менять в одном месте — тут.

## Файлы

### `app.json`
Главный конфиг: название, версия, пакет, порт, цвета, список языков.
Скрипт `build.sh` читает это и подставляет везде.

### `translations/`
Переводы для Android-приложения.

```
translations/
├── en/strings.xml   ← английский (дефолт)
├── ru/strings.xml   ← русский
└── be/strings.xml   ← белорусский
```

**Чтобы добавить новый язык**, например украинский:
1. Скопируй `en/strings.xml` → `uk/strings.xml`
2. Переведи строки
3. Добавь `"uk"` в `app.json` → `supported_languages`
4. Запусти `./build.sh` — скрипт сам скопирует в `values-uk/`

### `branding/`
- `countries.json` — названия стран на всех языках
